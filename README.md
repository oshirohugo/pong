# Pong game

This is a simple pong game. it works together with a pong server. If you want to see it live, go to this [url](http://www.students.ic.unicamp.br/~ra146446/pong)

![pong gif](pong.gif)

## Running app

Install dependencies:

```sh
yarn install
```

Run in dev mode:

```sh
yarn dev
```

Build:

```sh
yarn build
```
